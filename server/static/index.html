Blueprints API Documentation

GET /api/v1/blueprint

Retrieve a list of all blueprints.

GET /api/v1/blueprint/:id

Retrieve a specific blueprint by its ID.

Path Parameters:

  • id (string) - The ID of the blueprint.
POST /api/v1/blueprint

Create a new blueprint.

Request Body:

{ "name": "string", "version": "string", "author": "string", "blueprint_data": "object" }
PUT /api/v1/blueprint/:id

Update an existing blueprint by its ID.

Path Parameters:

  • id (string) - The ID of the blueprint.

Request Body: (Partial Update)

{ "name": "string", "version": "string", "author": "string", "blueprint_data": "object" }
DELETE /api/v1/blueprint/:id

Delete a blueprint by its ID.

Path Parameters:

  • id (string) - The ID of the blueprint.
